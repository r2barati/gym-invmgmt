gym-invmgmt

A Gymnasium-compatible benchmark for auditable inventory-management policy evaluation across optimization, heuristic, and learned controllers.

arXiv Paper

gym-invmgmt: An Open Benchmarking Framework for Inventory Management Methods

Reza Barati and Qinmin Vivian Hu. Submitted May 12, 2026. arXiv:2605.11355 [cs.LG].

The paper presents a shared evaluation contract for comparing inventory controllers across topology, demand, information-access, feasibility, shortage-treatment, and KPI settings.

arXiv PDF DOI

Environment Package

The standalone environment package provides the reusable CoreEnv, demand engine, topology parser, wrappers, visualization helpers, and Gymnasium registrations.

Install the published package with pip install gym-invmgmt.

PyPI GitHub Docs

Benchmark Artifacts

The companion repository contains paper experiments, benchmark agents, result tables, figure scripts, and reproducibility tooling.

Trained model checkpoints are hosted separately to keep the GitHub repositories lightweight.
